# Report

## Report an User

> Sample Request

```shell
curl "https://carikotak.com/api/report"
  -X "POST"
```

#### HTTP REQUEST

- `POST https://carikotak.com/api/report`

#### PARAMETERS
Parameter   |   |   Description
--------    | - |   -----------
target_id   | required  | target ID
target_type | required  | 'user'
title       | required  | report title
body        | required  | report body
