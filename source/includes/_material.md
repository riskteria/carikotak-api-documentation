# Material

## Get All Material

> Sample Request

```shell
curl "https://carikotak.com/api/material"
  -X "GET"
```

#### HTTP REQUEST
- `GET https://carikotak.com/api/material`

## Get a Material with Products

> Sample Request

```shell
curl "https://carikotak.com/api/material/33"
  -X "GET"
```

#### HTTP REQUEST
- `GET https://carikotak.com/api/material/{material_id}`
